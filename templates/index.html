📊 動画ログ照合システム

研修動画の視聴ログと出勤記録を照合し、勤務時間外視聴をチェックします

ログアウト
🎬 動画ログ 必須
📂
クリックまたはドラッグ&ドロップでファイルを選択
対応形式: Excel (.xlsx / .xls)
必要な列: 社員番号 / 氏名、日付、視聴開始時刻、視聴終了時刻、動画タイトル（進捗率は任意）
列名は日本語・英語どちらにも対応しています。
🗂 出勤記録 任意・複数可
📋
クリックまたはドラッグ&ドロップでファイルを選択
対応形式: Excel / CSV / PDF｜複数ファイル同時アップロード可
必要な列: 社員番号 / 氏名、日付、出勤時刻、退勤時刻
月次・日次など複数ファイルをまとめてアップロードできます。
📄 雇用契約書（所定勤務時間） 任意
📑
クリックまたはドラッグ&ドロップでファイルを選択
対応形式: Excel / CSV / PDF｜タイムカードがない従業員に使用
必要な列: 社員番号 / 氏名、所定始業時刻、所定終業時刻、勤務曜日（任意）
タイムカードがない従業員の視聴ログは、この所定勤務時間で判定します。
📊 凡例（出力レポートの色分け）
OK（勤務時間内）
契約時間内（タイムカードなし）
勤務時間外視聴
契約時間外視聴
出勤記録なし
人物不一致（要確認）
